mitonano_core

An honestly-scoped, actually-tested Rust crate for particle simulation with stochastic motion (SDE) and pharmacokinetic payload decay - the "Tier 1" foundation built and verified rather than just described.

Status: compiles clean (cargo build, zero warnings), all 18 unit tests passing, each checked against a closed-form analytical solution rather than just "runs without panicking."

cargo test # run all 18 tests cargo run --release --example run_simulation # see it in action 

What's here

sde.rs — Euler-Maruyama SDE integrator, generic over drift/diffusion functions. Tested against the closed-form stationary distribution of an Ornstein-Uhlenbeck process and against the zero-noise deterministic ODE limit.

pk.rs — One-compartment IV bolus and oral-absorption (Bateman function) pharmacokinetic models, integrated with RK4 and checked against their exact analytical solutions at multiple time points.

environment.rs — Bounded 3D domain with reflecting boundaries, including correctness for large overshoots (not just small ones).

agent.rs — A particle combining SDE-driven motion with exponential payload decay. Tested against the Einstein relation (variance = 2·D·t) as an independent check on the diffusion coefficient wiring, plus pure drift and boundary-containment tests.

engine.rs — Tick-loop running a population of agents, producing per-tick aggregate stats. Tested end-to-end against the analytical decay curve for the whole population, not just a single agent.

What's deliberately NOT here

No GPU layer, no RL taxonomy learning, no federated orchestration, no FEM tissue mechanics, no multi-object tracking. This crate is meant as a real foundation to build outward from incrementally - one tested feature at a time - not a recreation of the seven-pillar.

src/sde.rs

Euler-Maruyama SDE integrator. Correctness tests: OU-process stationary distribution (mean/variance vs. known closed form) and zero-noise deterministic ODE limit.

//! Generic Euler–Maruyama SDE integrator. //! //! Solves dX_t = mu(X_t, t) dt + sigma(X_t, t) dW_t //! //! This is deliberately generic over the drift/diffusion functions so it can //! drive both particle motion (agent.rs) and, if needed later, other //! stochastic quantities, without duplicating the integration logic. use rand_distr::{Distribution, Normal}; use rand::Rng; /// A single Euler–Maruyama step: X_{t+dt} = X_t + mu(X_t,t)*dt + sigma(X_t,t)*sqrt(dt)*Z /// where Z ~ N(0,1). pub fn em_step<F, G, R: Rng + ?Sized>( x: f64, t: f64, dt: f64, mu: F, sigma: G, rng: &mut R, ) -> f64 where F: Fn(f64, f64) -> f64, G: Fn(f64, f64) -> f64, { let normal = Normal::new(0.0, 1.0).expect("valid normal params"); let z: f64 = normal.sample(rng); x + mu(x, t) * dt + sigma(x, t) * dt.sqrt() * z } /// Integrates an SDE path over `n_steps` of size `dt`, returning the full path /// (including the initial value at index 0). pub fn integrate_path<F, G, R: Rng + ?Sized>( x0: f64, dt: f64, n_steps: usize, mu: F, sigma: G, rng: &mut R, ) -> Vec<f64> where F: Fn(f64, f64) -> f64, G: Fn(f64, f64) -> f64, { let mut path = Vec::with_capacity(n_steps + 1); let mut x = x0; let mut t = 0.0; path.push(x); for _ in 0..n_steps { x = em_step(x, t, dt, &mu, &sigma, rng); t += dt; path.push(x); } path } /// Ornstein-Uhlenbeck process parameters: dX_t = theta*(mu - X_t) dt + sigma dW_t /// Used both as a reusable "mean-reverting diffusion" building block (a /// reasonable model for a particle confined near a target site) and as the /// integrator's own correctness test, since its stationary distribution has /// a known closed form: X_inf ~ N(mu, sigma^2 / (2*theta)). #[derive(Debug, Clone, Copy)] pub struct OuParams { pub theta: f64, pub mu: f64, pub sigma: f64, } impl OuParams { pub fn drift(&self) -> impl Fn(f64, f64) -> f64 + '_ { move |x, _t| self.theta * (self.mu - x) } pub fn diffusion(&self) -> impl Fn(f64, f64) -> f64 + '_ { move |_x, _t| self.sigma } /// Theoretical stationary variance, for validating the integrator. pub fn stationary_variance(&self) -> f64 { self.sigma.powi(2) / (2.0 * self.theta) } } #[cfg(test)] mod tests { use super::*; use rand::SeedableRng; use rand::rngs::StdRng; /// Integrates many independent OU paths out to a time long enough to /// have equilibrated (t >> 1/theta), then checks the empirical mean and /// variance across paths against the known closed-form stationary /// distribution. This is the correctness check for the EM integrator /// itself, not just for OU specifically: if the stepping logic were /// wrong (e.g. dt vs sqrt(dt) swapped, or drift/diffusion mixed up), /// this test would fail. #[test] fn ou_process_converges_to_known_stationary_distribution() { let params = OuParams { theta: 2.0, mu: 5.0, sigma: 1.0 }; let dt = 0.001; // 1/theta = 0.5, so t_final = 8.0 is 16 relaxation times -> well equilibrated. let n_steps = 8_000; let n_paths = 2_000; let mut rng = StdRng::seed_from_u64(42); let mut finals = Vec::with_capacity(n_paths); for _ in 0..n_paths { let path = integrate_path(0.0, dt, n_steps, params.drift(), params.diffusion(), &mut rng); finals.push(*path.last().unwrap()); } let mean: f64 = finals.iter().sum::<f64>() / n_paths as f64; let var: f64 = finals.iter().map(|x| (x - mean).powi(2)).sum::<f64>() / (n_paths as f64 - 1.0); let expected_mean = params.mu; let expected_var = params.stationary_variance(); // 1.0^2 / (2*2.0) = 0.25 assert!( (mean - expected_mean).abs() < 0.05, "mean {mean} too far from theoretical {expected_mean}" ); assert!( (var - expected_var).abs() < 0.05, "var {var} too far from theoretical {expected_var}" ); } /// Sanity check with zero diffusion: the SDE degenerates to a plain ODE, /// dX/dt = theta*(mu - X), whose exact solution is /// X(t) = mu + (x0 - mu) * exp(-theta*t). Confirms the drift term alone /// integrates correctly, independent of any randomness. #[test] fn zero_noise_matches_deterministic_ode_solution() { let params = OuParams { theta: 1.5, mu: 3.0, sigma: 0.0 }; let dt = 0.0001; let n_steps = 50_000; // t_final = 5.0 let mut rng = StdRng::seed_from_u64(1); let path = integrate_path(10.0, dt, n_steps, params.drift(), params.diffusion(), &mut rng); let t_final = dt * n_steps as f64; let expected = params.mu + (10.0 - params.mu) * (-params.theta * t_final).exp(); assert!( (path.last().unwrap() - expected).abs() < 1e-3, "got {}, expected {}", path.last().unwrap(), expected ); } } 

src/pk.rs

One-compartment IV bolus and oral-absorption (Bateman function) pharmacokinetic models. Correctness tests: RK4 integration checked against exact analytical solutions at multiple time points, plus a half-life identity check and a mass-balance sanity check.

//! Pharmacokinetic compartment models. //! //! These are ODEs (deterministic), integrated with a simple RK4 stepper for //! accuracy, and validated against their known closed-form solutions. //! Kept deliberately to the two standard textbook one-compartment models //! rather than anything exotic, since those are the cases with an //! unambiguous "correct answer" to test against. /// One-compartment IV bolus model: dC/dt = -k_e * C /// Exact solution: C(t) = C0 * exp(-k_e * t) #[derive(Debug, Clone, Copy)] pub struct IvBolusPk { pub c0: f64, pub k_elim: f64, } impl IvBolusPk { pub fn analytical(&self, t: f64) -> f64 { self.c0 * (-self.k_elim * t).exp() } fn derivative(&self, c: f64) -> f64 { -self.k_elim * c } /// RK4-integrated concentration path over `n_steps` of size `dt`. pub fn integrate(&self, dt: f64, n_steps: usize) -> Vec<f64> { let mut c = self.c0; let mut path = Vec::with_capacity(n_steps + 1); path.push(c); for _ in 0..n_steps { let k1 = self.derivative(c); let k2 = self.derivative(c + 0.5 * dt * k1); let k3 = self.derivative(c + 0.5 * dt * k2); let k4 = self.derivative(c + dt * k3); c += (dt / 6.0) * (k1 + 2.0 * k2 + 2.0 * k3 + k4); path.push(c); } path } } /// One-compartment oral-absorption model (first-order absorption + elimination): /// dA/dt = -k_a * A (amount remaining in gut depot) /// dC/dt = (k_a * A) / V - k_e * C (plasma concentration) /// Exact solution (Bateman function), valid for k_a != k_e: /// C(t) = (Dose * k_a) / (V * (k_a - k_e)) * (exp(-k_e*t) - exp(-k_a*t)) #[derive(Debug, Clone, Copy)] pub struct OralAbsorptionPk { pub dose: f64, pub volume: f64, pub k_absorb: f64, pub k_elim: f64, } impl OralAbsorptionPk { pub fn analytical(&self, t: f64) -> f64 { let ka = self.k_absorb; let ke = self.k_elim; (self.dose * ka) / (self.volume * (ka - ke)) * ((-ke * t).exp() - (-ka * t).exp()) } fn derivatives(&self, a: f64, c: f64) -> (f64, f64) { let da = -self.k_absorb * a; let dc = (self.k_absorb * a) / self.volume - self.k_elim * c; (da, dc) } /// RK4-integrated (depot_amount, plasma_concentration) path. pub fn integrate(&self, dt: f64, n_steps: usize) -> Vec<(f64, f64)> { let mut a = self.dose; let mut c = 0.0; let mut path = Vec::with_capacity(n_steps + 1); path.push((a, c)); for _ in 0..n_steps { let (ka1, kc1) = self.derivatives(a, c); let (ka2, kc2) = self.derivatives(a + 0.5 * dt * ka1, c + 0.5 * dt * kc1); let (ka3, kc3) = self.derivatives(a + 0.5 * dt * ka2, c + 0.5 * dt * kc2); let (ka4, kc4) = self.derivatives(a + dt * ka3, c + dt * kc3); a += (dt / 6.0) * (ka1 + 2.0 * ka2 + 2.0 * ka3 + ka4); c += (dt / 6.0) * (kc1 + 2.0 * kc2 + 2.0 * kc3 + kc4); path.push((a, c)); } path } } #[cfg(test)] mod tests { use super::*; #[test] fn iv_bolus_matches_exponential_decay_closed_form() { let pk = IvBolusPk { c0: 100.0, k_elim: 0.3 }; let dt = 0.01; let n_steps = 1000; // t_final = 10.0 let path = pk.integrate(dt, n_steps); for (i, &c) in path.iter().enumerate() { let t = i as f64 * dt; let expected = pk.analytical(t); assert!( (c - expected).abs() < 1e-6, "at t={t}: numeric={c}, analytical={expected}" ); } } #[test] fn iv_bolus_half_life_is_correct() { // Half-life = ln(2)/k_elim is a standard PK identity; check the // integrator reproduces it, not just the raw ODE formula. let k_elim = 0.1386_29; // ln(2)/5.0 -> half-life should be ~5.0 let pk = IvBolusPk { c0: 50.0, k_elim }; let path = pk.integrate(0.001, 5000); // t_final = 5.0 let c_final = *path.last().unwrap(); assert!( (c_final - 25.0).abs() < 0.01, "expected concentration to halve by t=5.0, got {c_final}" ); } #[test] fn oral_absorption_matches_bateman_function() { let pk = OralAbsorptionPk { dose: 200.0, volume: 10.0, k_absorb: 1.2, k_elim: 0.2 }; let dt = 0.001; let n_steps = 20_000; // t_final = 20.0 let path = pk.integrate(dt, n_steps); // Spot-check several time points against the closed-form Bateman // solution, including near the concentration peak where numerical // error tends to concentrate. for &i in &[100usize, 1_000, 2_000, 5_000, 10_000, 20_000] { let t = i as f64 * dt; let numeric = path[i].1; let expected = pk.analytical(t); assert!( (numeric - expected).abs() < 1e-3, "at t={t}: numeric={numeric}, analytical={expected}" ); } } #[test] fn oral_absorption_mass_balance_holds() { // Sanity check independent of the closed form: at t=0 all dose is // in the depot and none in plasma. let pk = OralAbsorptionPk { dose: 200.0, volume: 10.0, k_absorb: 1.2, k_elim: 0.2 }; Bounded 3D domain with reflecting boundaries. Correctness tests: identity for in-bounds points, correct reflection for both small and large (multi-width) overshoots.

//! Bounded simulation domain. //! //! Deliberately simple: an axis-aligned box with reflecting boundaries. //! No voxel concentration grid yet — that's a real feature (needed for //! agent-agent interaction via local concentration fields) but it's the //! next increment on top of this, not part of getting the core loop right. #[derive(Debug, Clone, Copy)] pub struct Bounds { pub min: [f64; 3], pub max: [f64; 3], } impl Bounds { pub fn new(min: [f64; 3], max: [f64; 3]) -> Self { assert!(min[0] < max[0] && min[1] < max[1] && min[2] < max[2], "invalid bounds"); Self { min, max } } pub fn contains(&self, p: [f64; 3]) -> bool { (0..3).all(|i| p[i] >= self.min[i] && p[i] <= self.max[i]) } /// Reflects a position back into bounds on each axis independently. /// Simple fold-reflection: works correctly even if a step overshoots by /// more than one box width, by repeated folding rather than a single /// mirror (a single mirror is only correct for small overshoots). pub fn reflect(&self, p: [f64; 3]) -> [f64; 3] { let mut out = [0.0; 3]; for i in 0..3 { let lo = self.min[i]; let hi = self.max[i]; let width = hi - lo; let mut x = (p[i] - lo).rem_euclid(2.0 * width); if x > width { x = 2.0 * width - x; } out[i] = lo + x; } out } } #[cfg(test)] mod tests { use super::*; #[test] fn reflect_is_identity_inside_bounds() { let b = Bounds::new([0.0, 0.0, 0.0], [10.0, 10.0, 10.0]); let p = [5.0, 3.0, 7.0]; let r = b.reflect(p); assert!((r[0] - p[0]).abs() < 1e-9); assert!((r[1] - p[1]).abs() < 1e-9); assert!((r[2] - p[2]).abs() < 1e-9); } #[test] fn reflect_brings_small_overshoot_back_in_bounds() { let b = Bounds::new([0.0, 0.0, 0.0], [10.0, 10.0, 10.0]); // Overshoot past max on x by 1.0 -> should reflect to 9.0 let r = b.reflect([11.0, 5.0, 5.0]); assert!(b.contains(r)); assert!((r[0] - 9.0).abs() < 1e-9); } #[test] fn reflect_handles_large_overshoot_past_multiple_widths() { let b = Bounds::new([0.0, 0.0, 0.0], [10.0, 10.0, 10.0]); // 25.0 is 2.5 widths past min -> must still land in [0,10] let r = b.reflect([25.0, -13.0, 5.0]); assert!(b.contains(r), "reflected point {:?} not in bounds", r); } #[test] fn reflect_never_produces_out_of_bounds_point_across_many_random_offsets() { let b = Bounds::new([-2.0, 0.0, 1.0], [3.0, 8.0, 6.0]); for i in -50..50 { let offset = i as f64 * 1.37; let p = [1.0 + offset, 4.0 + offset, 3.0 + offset]; let r = b.reflect(p); assert!(b.contains(r), "offset {offset}: reflected {:?} not in bounds", r); } } } 

src/agent.rs

A particle combining SDE-driven motion with exponential PK-based payload decay. Correctness tests: exact first-order decay, zero-motion stasis, pure-drift displacement, boundary containment under strong diffusion, and the Einstein relation (Var = 2·D·t) as an independent check on diffusion wiring.

//! A single simulated particle/agent: position evolves by Brownian motion //! with drift (via sde.rs), confined to the environment's bounds, while its //! carried payload amount decays according to a PK elimination model. use crate::environment::Bounds; use crate::sde::em_step; use rand::Rng; #[derive(Debug, Clone, Copy)] pub struct AgentParams { /// Constant drift velocity, e.g. from fluid flow or a chemotactic bias. pub drift: [f64; 3], /// Diffusion coefficient (isotropic Brownian motion strength). pub diffusion_coeff: f64, /// First-order elimination rate for the carried payload. pub k_elim: f64, } #[derive(Debug, Clone, Copy)] pub struct Agent { pub position: [f64; 3], pub payload: f64, } impl Agent { pub fn new(position: [f64; 3], initial_payload: f64) -> Self { Self { position, payload: initial_payload } } /// Advances the agent by one tick: Euler-Maruyama step per axis for /// position, reflected into bounds, and an RK4-equivalent-accuracy /// exact exponential update for the payload (first-order decay has a /// closed form, so we use it directly rather than re-deriving it via a /// generic ODE stepper -- no reason to introduce numerical error where /// an exact update is available and cheap). pub fn step<R: Rng + ?Sized>(&mut self, params: &AgentParams, bounds: &Bounds, dt: f64, rng: &mut R) { let mut new_pos = [0.0; self.payload *= (-params.k_elim * dt).exp(); } } #[cfg(test)] mod tests { use super::*; use rand::SeedableRng; use rand::rngs::StdRng; #[test] fn payload_decays_exactly_per_first_order_kinetics() { let bounds = Bounds::new([-1000.0, -1000.0, -1000.0], [1000.0, 1000.0, 1000.0]); src/engine.rs

Tick-loop engine running a population of agents, producing per-tick aggregate stats. Correctness tests: population-mean decay matches single-agent analytical decay end-to-end, history length correctness, and no-escape-from-bounds over a full run.

//! Tick-loop engine: advances a population of agents through the //! environment for a fixed number of steps and records aggregate stats //! each tick, for downstream analysis or plotting. use crate::agent::{Agent, AgentParams}; use crate::environment::Bounds; use rand::Rng; #[derive(Debug, Clone, Copy)] pub struct TickStats { pub tick: u64, pub mean_payload: f64, pub total_payload: f64, pub mean_position: [f64; 3], } pub struct Engine { pub agents: Vec<Agent>, pub params: AgentParams, pub bounds: Bounds, pub dt: f64, } impl Engine { pub fn new(agents: Vec<Agent>, params: AgentParams, bounds: Bounds, dt: f64) -> Self { Self { agents, params, bounds, dt } } fn stats_for_tick(&self, tick: u64) -> TickStats { let n = self.agents.len() as f64;
